# Renewable Energy Distribution Dashboard

An interactive data visualisation dashboard built with D3.js, displaying UK renewable energy trends from 1990–2023.

## Features
- Grouped bar chart, treemap, line chart, and donut chart
- Cross-chart linked highlighting
- Data sourced from UK government energy statistics

## Technologies
- D3.js
- HTML / CSS / JavaScript
